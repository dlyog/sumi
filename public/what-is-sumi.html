What is Sumi?

Sumi is an AI Learning Companion built into an interactive learning application. She understands the current screen, explains concepts, demonstrates approved actions, and helps learners learn by doing.

How to use Sumi

  1. Sign in and select the Sumi icon.
  2. Ask a question or describe what you want to try.
  3. Make a prediction when Sumi asks one.
  4. Watch the registered action happen in the application.
  5. Inspect the verified result and try the experiment yourself.

What Sumi is not

Sumi is not a general-purpose assistant and does not execute arbitrary code. Each learning site registers its own screens, concepts, explanations, and safe actions through the Sumi SDK and CLI.

Learning by doing

1StopQuantum uses Sumi to make quantum computing approachable: learners describe circuits in natural language, run local simulations, inspect states and measurements, and connect every answer to visible evidence.
