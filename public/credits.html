Benchmark data and reference projects

  • Unitary Foundation metriq-data provides the attributable benchmark dataset used by the local Benchmark workspace. Its data is published under CC BY 4.0; 1StopQuantum records source paths and transformation details.
  • Unitary Foundation metriq-gym provides the open framework and benchmark definitions used to acquire Metriq results.

The repository file docs/METRIQ_ATTRIBUTION.md records the bundled snapshot, source revision, and transformations.

Quantum and scientific foundations

  • Qiskit and Qiskit Aer support circuit compilation and local simulation.
  • Cirq provides the alternate circuit simulator and framework comparison.
  • D-Wave Neal supports the local simulated-annealing lesson.
  • RDKit supports educational molecule parsing, rendering, and descriptors.

Application and visualization foundations

  • FastAPI, PostgreSQL, and Model Context Protocol Python SDK support the local API, persistence, and integrations.
  • D3, Three.js, and Monaco Editor support interactive learning and circuit authoring.
  • Playwright supports browser accessibility, responsive-layout, and regression verification.

Additional software dependencies

The application also relies on Pydantic, Uvicorn, NumPy, SciPy, Pillow, PyYAML, jsonschema, psycopg, cryptography, OpenAI Python, esbuild, and TypeScript. The repository lockfiles and requirements file are the authoritative version inventory.

Maintainer media tools

  • ComfyUI and the locally installed DreamShaper checkpoint were used to pre-author lesson images.
  • Kokoro was used to pre-author bundled lesson and workspace narration.

These authoring systems are not learner-facing runtime dependencies. Accepted media is reviewed, saved locally, and disclosed with provenance.
